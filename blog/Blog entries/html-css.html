What are the best practices associated with using classes vs. ids?

Week 1 Technical Blog

17 June 2025

When to use Classes

Classes are a type of attribute that can be applied to HTML elements. The cool thing about the class attribute is that you can apply it multiple times throughout a single HTML page. Class attributes should always be your first go to option when you are giving attributes to elements... This is because you can always override a class attribute by using an ID attribute. If you wanted certain elements to have the same style you would use a class attribute - becuase you can use the same class throughout different elements, across a single document.

When to use Id's

Id's are another type of attribute that can be applied to HTML elements. One thing about the Id attribute is that you can only use an Id once, in one location, throughout a single HTML page. Id attributes can be used for specific styling in a webpage, if you want to style one element and override previous class attribute styling for that element. A cool way to use an Id attribute is if you want to include an anchor tag in your webpage. You can use the Id attribute as a reference point for text that is up and down a webpage (when you use the scroll bar). I will have a demonstration of this at the end of this Blog post to show an example of an anchor tag being used in a webpage...

Classes and Id's in CSS

When working with class and Id attributes in CSS, the two different attributes act the same when CSS applies the styling. Technically you could use either attribute when applying styling in CSS... However, as stated above it is always good practice to use the class attribute first - you can always override it later with an Id attribute if you need. In order to select a class when we are styling in CSS we can use the class selector. The class selector is a period '.' + the class name 'class-name-here' + a space ' ' and + some curly brackets '{}'. Similar to class selectors, we also have Id selectors, we can use these to acess a specific Id on a page. The Id selector is a hash key '#' + the Id name 'Id-name-here' + a space ' ' + some curly brackets '{}'.
