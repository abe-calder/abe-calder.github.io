A time I was blocked on a simple problem

During the Javascript Kata FlipFlop challenge, I was blocked on the challenge as I couldn't get the test to pass and wasn't sure what was wrong. I had my FlipFlop function with if statements that used the remainder operator to check if a number was divisible by 15, 5, or 3. I had a feeling that the if statements were correct and I was getting confused why the tests weren't passing. I kept looking at the code and wondering why I wasn't working as it should, it was a bit frustrating and felt like I couldn't figure it out. But, after trying lots of different things over and over it ended up passing. I was putting the if statements, which check if a number is divisible by, 3, 5, or 15, in the wrong order. Since 15 is divisible by 3 and 5, it needs to be run first in the code, then the condition that check if a number is divisible by 5 is next, and then 3.
